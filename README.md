A lightweight and efficient CLI tool written in Go to analyze word frequency from text files.

---

## ⚙️ Features

- 📂 Reads input file via CLI argument  
- 🔤 Cleans & normalizes text (case-insensitive, removes punctuation)  
- 📊 Counts word frequency using hash maps  
- 📈 Sorts results by frequency (descending)  
- ⚡ Displays execution time  
- 🧠 Tracks memory usage  

---

## 🚀 Usage

```bash
go run main.go <filename.txt>
